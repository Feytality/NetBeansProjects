To run this application

1) Add this project to Netbeans
2) Right click on the project, and select 'Run'
3) It will take you to a page where you can test 3 functionalities
	a) Get popular bookmarks : click the button "Get Popular Bookmarks" and a list will be populated on top of the button.
	b) Get recent bookmarks : click the button "Get Recent Bookmarks" and a list will be populated on top of the button.
	c) Get popular bookmarks : Set a username in the text box (ex: serviceavenir), click the button "Get User Bookmarks" and a list will be populated on top of the button.
